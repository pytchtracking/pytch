Pytch Vocal Trainer
===============

## Download and Installation

Clone the project
```
git clone https://github.com/pytchtracking/pytch
cd pytch
```

Install pytch:
```
pip3 install .
```

For development purposes, install pytch with:
```
pip3 install -e .
```

## Run
Open a terminal, type
```
pytch
```
hit return and sing!
